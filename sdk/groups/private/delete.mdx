---
title: "groups.private.delete"
description: "Pinata SDK groups.private.delete removes a private file group from your account. The files inside the group are not unpinned by this call."
---

Delete a private group

## Usage

```typescript
import { PinataSDK } from "pinata";

const pinata = new PinataSDK({
  pinataJwt: process.env.PINATA_JWT!,
  pinataGateway: "example-gateway.mypinata.cloud",
});

const groups = await pinata.groups.private.delete({
	groupId: "3778c10d-452e-4def-8299-ee6bc548bdb0",
});
```

## Returns

```typescript
OK
```

## Parameters

### groupId

- Type: `string`

ID for the target Group
