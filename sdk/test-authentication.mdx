---
title: "testAuthentication SDK method"
description: "Use the testAuthentication SDK method to verify that your Pinata JWT works against the configured network before running real uploads."
---

Tests authentication with the current `PINATA_JWT`

## Usage

```typescript
import { PinataSDK } from "pinata";

const pinata = new PinataSDK({
  pinataJwt: process.env.PINATA_JWT!,
  pinataGateway: "example-gateway.mypinata.cloud",
});

const auth = await pinata.testAuthentication()
```

## Returns

```typescript
type AuthTestResponse = {
  message: string;
};
```
