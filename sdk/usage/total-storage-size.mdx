---
title: "totalStorageSize"
description: "Pinata SDK totalStorageSize returns the total storage in bytes that pinned files use on your Pinata account across public and private IPFS."
---

List total storage in bytes

## Usage

```typescript
import { PinataSDK } from "pinata";

const pinata = new PinataSDK({
  pinataJwt: process.env.PINATA_JWT!,
  pinataGateway: "example-gateway.mypinata.cloud",
});

const pinnedFiles = await pinata.usage.totalStorageSize();
```

## Returns

- Type: `number`

Total storage used in bytes
