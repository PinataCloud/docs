---
title: "pinnedFileCount"
description: "Pinata SDK pinnedFileCount returns the total number of files currently pinned to your Pinata account across public and private IPFS networks."
---

List total number of files currently pinned

## Usage

```typescript
import { PinataSDK } from "pinata";

const pinata = new PinataSDK({
  pinataJwt: process.env.PINATA_JWT!,
  pinataGateway: "example-gateway.mypinata.cloud",
});

const pinnedFiles = await pinata.usage.pinnedFileCount();
```

## Returns

- Type: `number`

Number of total files pinned
