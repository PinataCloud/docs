Skills are code bundles that extend what your agent can do. Browse and install skills at [agents.pinata.cloud/skills](https://agents.pinata.cloud/skills).

## Browse Skills

Go to [agents.pinata.cloud/skills](https://agents.pinata.cloud/skills) to see available skills:

| Tab | Description |
| --- | ----------- |
| **All \| Installed** | Skills in your library |
| **ClawHub Community** | Community marketplace |

### Official Pinata Skills

Look for the **BY PINATA** and **VERIFIED** badges:

| Skill | Description |
| ----- | ----------- |
| **@pinata/api** | Pinata IPFS API for file storage, gateways, and vector search |
| **@pinata/erc-8004** | Register AI agents on-chain |
| **@pinata/memory-salience** | Salient memory with novelty, retention, and decay |
| **@pinata/paraspace** | PARA-based workspace organization |
| **@pinata/sqlite-sync** | SQLite with Pinata backup and versioning |

### Community Skills

Browse the **ClawHub Community** tab for community-created skills. Each shows download count and star rating.

## Install a Skill

1. Click a skill card to view details
2. Review the description and required secrets
3. Click **Install**

The skill is added to your library and available to attach to any agent.

## Attach a Skill to an Agent

**During creation:** Add skills in Step 2 (Workspace) of the setup wizard.

**After deployment:** Open your agent → **Skills** → Click **Add** → Select from your library.

Skills are copied into the agent's workspace. Detaching removes the files but keeps the skill in your library.

## Create a Custom Skill

1. Click **Upload Skill**
2. Select a folder containing your skill files

Your folder must include `SKILL.md` with YAML frontmatter:

```markdown
---
name: my-skill
description: What this skill does
---

# My Skill

Instructions for using this skill.
```

<Note>
  Skills are uploaded to the public IPFS network.
</Note>

## Skill Details

Click any skill to view:

- **Description** — Documentation and usage instructions
- **CID** — IPFS content identifier
- **Required Secrets** — Environment variables needed
- **Used By** — Agents using this skill

## Using the CLI

```bash
# List skills in your library
pinata agents skills list

# Upload a skill
pinata agents skills upload <path>
```

## Constraints

| Limit | Value |
| ----- | ----- |
| Skills per agent | 10 |
| Skill name | Letters, numbers, hyphens, underscores |
