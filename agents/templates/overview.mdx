Templates are pre-configured agent packages with bundled [skills](/agents/skills), required [secrets](/agents/secrets), and default settings. Browse at [agents.pinata.cloud/templates](https://agents.pinata.cloud/templates) or use the [CLI](/tools/cli/agents#templates).

## Browsing Templates

Filter templates using the tabs:

| Tab | Description |
| --- | ----------- |
| **All** | All available templates |
| **General** | General-purpose agents |
| **My Submissions** | Templates you've submitted |

## Template Details

Click a template to view:

- **Description** — What it does
- **Category** and **Tags**
- **Version**
- **Required Setup** — Secrets you need to provide

### Required Setup

Templates list the secrets needed before deployment. Example for Telegram Bot:

| Secret | Description |
| ------ | ----------- |
| `ANTHROPIC_API_KEY` | Anthropic API key for Claude |
| `TELEGRAM_BOT_TOKEN` | Token from BotFather |

Add these to your [Secrets vault](/agents/secrets) before deploying.

## Deploying

1. Select a template
2. Review **Required Setup**
3. Add any missing secrets
4. Click **Deploy This Agent**
