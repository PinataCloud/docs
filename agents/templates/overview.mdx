Templates are pre-configured agent packages with bundled [skills](/agents/skills), required [secrets](/agents/secrets), and default settings. Deploy a template to get a working agent in minutes.

## Browse Templates

Go to [agents.pinata.cloud/templates](https://agents.pinata.cloud/templates) to browse available templates.

| Tab | Description |
| --- | ----------- |
| **All** | All published templates |
| **General** | General-purpose agents |
| **My Submissions** | Templates you've created |

Click any template to see its details: description, category, tags, and required secrets.

## Deploy a Template

1. Click a template to open its details
2. Review the **Required Setup** section — these are secrets you'll need
3. Click **Deploy This Agent**
4. Enter a name for your agent
5. Add any missing secrets when prompted
6. Click **Deploy**

Your agent will provision and you'll be redirected to its detail page.

### Required Secrets

Most templates need at least an LLM provider key. The template details show exactly what's required:

| Secret | Description |
| ------ | ----------- |
| `ANTHROPIC_API_KEY` | Anthropic API key for Claude |
| `TELEGRAM_BOT_TOKEN` | Token from BotFather (for Telegram templates) |

Add secrets to your [vault](/agents/secrets) before deploying, or you'll be prompted during setup.

## Using the CLI

You can also deploy templates via the [CLI](/tools/cli/agents#templates):

```bash
# List available templates
pinata agents templates list

# Deploy a template
pinata agents create --template <template-id>
```
