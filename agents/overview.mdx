Pinata Agents are hosted [OpenClaw](https://openclaw.org) instances. Each agent runs in an isolated container with its own workspace, skills, and secrets.

<Note>
  Agents require a paid Pinata plan. [Upgrade here](https://app.pinata.cloud/billing).
</Note>

## Quick Start

### 1. Connect an LLM Provider

Your agent needs an LLM provider to run. Go to [agents.pinata.cloud/secrets](https://agents.pinata.cloud/secrets) and click **Connect** on one of the provider cards:

- **Anthropic** → `ANTHROPIC_API_KEY`
- **OpenAI** → `OPENAI_API_KEY`
- **OpenRouter** → `OPENROUTER_API_KEY`

Or use the command palette (**⌘K**) and search for "Connect Anthropic" (or your preferred provider).

### 2. Create an Agent

Go to [agents.pinata.cloud/agents](https://agents.pinata.cloud/agents) and click **Create Agent**.

**Option A: Start from a Template**

Select a pre-configured [template](/agents/templates/overview) to skip manual setup. Templates include skills, required secrets, and default settings.

**Option B: Configure Manually**

1. **Identity** — Choose a personality preset (Atlas, Nova, Sage) or create a custom one. Enter a name and optional description.
2. **Workspace** — The default workspace includes Node.js, Python, and common tools. Optionally add [skills](/agents/skills).
3. **Connect** — Select your LLM provider (shows "Available" when the key exists). Add any additional secrets. Optionally configure [channels](/agents/channels).
4. **Deploy** — Click **Deploy Agent**. Once provisioning completes, you're redirected to your agent's detail page.

### 3. Start Chatting

Your agent is ready. Use the **Chat** interface to start a conversation.

## Agent Detail Page

Click any agent to open its detail page. The sidebar provides access to all features:

| Section | Purpose |
| ------- | ------- |
| **Chat** | Conversation interface |
| **Channels** | Messaging platform connections |
| **Snapshots** | Workspace versions and git access |
| **Skills** | Attached skill packages |
| **Secrets** | Environment variables |
| **Models** | LLM provider configuration |
| **Routes** | Port forwarding and domains |
| **Tasks** | Scheduled jobs |

### Dev Tools

| Tool | Purpose |
| ---- | ------- |
| **Console** | Terminal access to your agent's container |
| **Logs** | Real-time logs with search and level filters |
| **Danger** | Settings, paired devices, restart, and delete |

### Git Access

Clone and edit your agent's workspace directly:

```bash
git clone https://agents.pinata.cloud/v0/agents/{agentId}/git {name}
```

Click **Copy with Token** in the Snapshots section to include authentication.

## Using the CLI

Everything you can do in the UI is also available via the [CLI](/tools/cli/agents):

```bash
# List your agents
pinata agents list

# Create an agent
pinata agents create

# Get agent details
pinata agents get <agent-id>
```
