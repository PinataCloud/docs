## **What is Pinata?**

Pinata makes decentralized file uploads _simple_ for developers who need fast, scalable, and zero-setup solutions. Our API and SDK make it as easy as one line of code to upload or retrieve files with IPFS.

**Copy to Upload Files**

```
const { cid } = await pinata.upload.public.file(file)

// Your CID
```

**Copy to Retrieve Files**

```
const { data, contentType } = await pinata.gateways.public.get(  
"Paste your CID"
)
```

Pinata helps developers, from university hackers to major Crypto teams, easily build on IPFS. Whether you’re minting NFTs, deploying a decentralized app, or experimenting at a hackathon, we’ve got you covered.

## **🛠️ How to Use Pinata — Step-by-Step Setup**

### 1. **Create Your Free Account**

Head to https://app.pinata.cloud/auth/sign-up and sign up.

No credit card. No blockchain wallet required.

### 2. **Upload a File**

Drag and drop your file right from your desktop into your dashboard.

You’ll instantly get a **CID**—your file’s unique fingerprint on IPFS.

### 3. **Retrieve It Anywhere**

Pop your CID into our gateway URL:

```
<https://gateway.pinata.cloud/ipfs/[YOUR_CID]>
```

Your file is now live on the decentralized web.

### 4. **Try It in Code**

Use our SDK or API to upload and retrieve programmatically.

It’s available in JavaScript and other languages.

### 5. **Build Something Cool**

Pinata is perfect for:

- NFT metadata and media
- Decentralized apps (dApps)
- Games and creative projects
- File storage for hackathon prototypes
